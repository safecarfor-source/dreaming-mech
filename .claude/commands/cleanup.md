# 규칙/스킬 정리

rules/와 .claude/skills/ 파일들을 점검하고 정리한다.

## 실행 방법

### 1. 규칙 점검
- 모든 `rules/*.md` 파일 읽기
- 모순되는 규칙 발견 시 보고
- 더 이상 유효하지 않은 규칙 식별
- `tasks/lessons.md`에서 규칙화할 만한 새 패턴 추출

### 2. 스킬 점검
- 모든 `.claude/skills/*.md` 파일 읽기
- 사용되지 않는 스킬 식별
- 개선 가능한 스킬 제안

### 3. 에이전트 점검
- 모든 `.claude/agents/*.md` 파일 읽기
- 역할 중복 확인
- 모델 설정 확인 (sonnet 통일 여부)

### 4. 커맨드 점검
- 모든 `.claude/commands/*.md` 파일 읽기
- 사용 빈도 낮은 커맨드 식별
- 통합 가능한 커맨드 제안

## 출력 형식
| 파일 | 상태 | 제안 |
|------|------|------|
| rules/X.md | 정상/모순/폐기 | 조치 내용 |
| skills/Y.md | 정상/개선/삭제 | 조치 내용 |

## 규칙
- 삭제는 제안만 하고 대장님 승인 후 실행
- 수정도 내용을 보여주고 승인 후 적용
