## 파이프라인 (7단계)

1. **영상 확보** — YouTube URL 또는 로컬 파일
2. **오디오 추출** — FFmpeg
3. **자막 생성** — Whisper (분석용)
4. **AI 분석** — Claude 2단계 (하이라이트 + 문구)
5. **클립 조립** — 검증 + 정렬
6. **렌더링** — FFmpeg 블랙바 + 훅 타이틀 + LUFS
7. **결과 저장** — MP4 + 썸네일 JPG
