# 꿈꾸는정비사 — 기능 아이디어 모음

> 음성 메모 및 대화에서 수집한 아이디어. 구현 시 참고.

---

## 1. 소비자-정비소 직접 소통 채널
- 소비자가 정비소에 직접 연락할 수 있는 방법 필요
- 전화번호는 민감 → **카카오 오픈채팅** 등 대안 활용
- 정비사가 적극적으로 문제 해결 → 신뢰 구축
- **상태**: 아이디어

## 2. 지역 기반 정비소 매칭 + 지도
- 자신의 지역 내 정비소 검색
- **지도에 위치 마킹** (카카오맵/네이버맵)
- 지역별 필터링
- **상태**: 아이디어

## 3. 견적 프로그램 웹 구현
- 현재 사용 중인 견적 프로그램을 웹에서 구현
- 소비자도 확인 가능한 형태
- **상태**: 아이디어

## 4. 정비소 데이터 확보
- 정비소 데이터 소스 조사
- 비용 확인 후 활용 여부 결정
- 공공 데이터 API, 크롤링 등 방법 검토
- **상태**: 조사 필요

## 5. 타이어 문의 가이드 영상
- 소비자가 타이어 문의 시 **사진 어떻게/왜 찍는지** 짧은 유튜브 영상 제작
- 문의 페이지에 영상 임베드
- 진입장벽 제거 → 부담 없이 문의 가능
- **상태**: 아이디어
