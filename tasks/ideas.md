# 꿈꾸는정비사 — 기능 아이디어 모음

> 음성 메모에서 수집한 아이디어. 구현 시 Claude에게 이 문서를 참조하도록 지시할 것.

---

## 플랫폼 3대 주체

| 주체 | 역할 | 전용 페이지 |
|------|------|------------|
| **관리자 (대표)** | 전체 운영, 전략 수립, 데이터 관리 | 관리자 대시보드 |
| **소비자** | 정비소 검색, 문의, 견적 요청 | 소비자 페이지 |
| **정비사** | 고객 관리, 견적 응답, 문의 처리 | 정비사 페이지 |

---

## 1. 소비자-정비소 직접 소통 채널
- 소비자가 정비소에 **직접 연락**할 수 있는 방법 필요
- 전화번호는 민감 → **카카오 오픈채팅** 등 대안 우선 활용
- 정비사가 소비자 문제를 **적극적으로 해결**하는 구조 → 신뢰 구축
- 정비사가 먼저 소비자에게 다가가는 방식

## 2. 지역 기반 정비소 매칭 + 지도
- 자신의 지역 내 정비소 검색
- **지도에 위치 마킹** (카카오맵/네이버맵)
- 지역별 필터링
- 지역 특성 고려 (예: 운정 같은 곳은 정비소 접근이 어려운 지역도 있음)

## 3. 견적 프로그램 웹 구현
- 현재 사용 중인 견적(극동) 프로그램을 **웹에서 구현**
- 소비자도 확인 가능한 형태
- 정비사가 견적 작성 → 소비자에게 공유

## 4. 정비소 데이터 확보
- 정비소 데이터 소스 조사 (공공 데이터 API, 크롤링 등)
- 비용 확인 후 비싸지 않으면 활용
- 데이터 정확성 검증 필요

## 5. 타이어 문의 가이드 영상
- 소비자가 타이어 문의할 때 **사진을 어떻게/왜 찍는지** 짧은 유튜브 영상 제작
- 문의 페이지에 영상 임베드
- 진입장벽 제거 → 부담 없이 문의 가능하게

## 6. 전쟁 지도 (전략 로드맵)
- 웹페이지를 바로 배포하지 말고, **단계별 로드맵을 시각적으로 표현**
- 마인드맵 형태로 각 기능의 활성화 상태를 한눈에 파악
- 3대 주체(관리자/소비자/정비사)의 **관계도** 포함
  - 소비자 페이지: 소비자가 이용하는 기능들
  - 정비사 페이지: 손님 관리, 견적 응답 등 정비사 기능들
  - 관리자 대시보드: 전체 운영 현황
- 각 기능이 오픈되면 로드맵에서 **활성화 표시**
- 대표가 전략을 세울 수 있는 **전쟁 지도** 역할

## 7. 조직도 시각화
- 조직도를 비주얼하게 만들되, **수평적 연결성**을 보여줄 것
- 각 부서별 진행률/완성도 표시
- **유튜브 부서**를 조직도에 포함
  - 유튜브 마케팅 담당
  - 콘텐츠 제작
  - 유튜브 성장 전략 연구

## 8. 유튜브 & SNS 마케팅 전략
- 유튜브, Meta, Threads, 인스타그램에서 **트래픽을 웹사이트로 유입**
- 유튜브 콘텐츠 기획:
  - 연령대별 맞춤 콘텐츠 (40대 이상 핵심 고객층)
  - 노인층을 위한 정비 관련 콘텐츠
  - 젊은층은 Threads 등 SNS로 유입 가능
- SNS 트렌드 분석 및 콘텐츠 방향 설정
- 유튜브에서 활동하는 정비사(유튜버 정비사)와 협업 가능성
- **감성적이고 깊이 있는 콘텐츠**가 중요

## 9. 온라인 진입장벽 해소
- 기존 정비소는 오프라인 중심 → 접근 어려움
- AI와 웹 기술을 활용해 **온라인으로 문의/신청** 가능하게
- 오프라인에서만 가능했던 것들을 온라인으로 전환

## 10. 인프라 안정성
- 안정적인 클라우드 인프라 사용 (AWS, Google Cloud 등)
- **제로 다운타임** 목표
- 대형 기업 수준의 안정성 확보
- 사용자 데이터 보안 유지
