> 작성일: 2026-02-26
> 목표: 무료로 웹사이트 디자인을 매우 깔끔하고 멋지게 만든다

## 1. 핵심 결론 (요약)

**Figma가 필요 없다.** 우리 상황에서 최적 경로:

```
[최적 경로] Tailwind 무료 컴포넌트 라이브러리 + CLAUDE.md 디자인 규칙 + Claude Code
= 비용 0원, 즉시 실행, 효과 최대
```

**이유:**
1. 대장님은 코딩을 안 하므로 Figma 디자인 → Claude Code 변환 과정이 불필요
2. Claude Code가 직접 코드를 작성하므로 중간 디자인 도구가 오히려 병목
3. 무료 Tailwind 컴포넌트 라이브러리가 이미 프로덕션 수준의 디자인 제공
4. 우리 CLAUDE.md에 이미 완벽한 디자인 규칙이 있음

**Figma 활용 시점:** 나중에 디자이너를 고용하거나, 복잡한 UI를 사전 설계할 때

## 4. 실행 계획 (3단계)

### 1단계: 즉시 디자인 개선 (소요: 반나절)
> Claude Code가 직접 코드 수정. 대장님은 localhost:3000에서 확인만.

**수정 대상:**
1. **메인페이지 폼 스타일 통일** — 모든 input/textarea 일관된 border + focus 스타일
2. **사장님 로그인 배경** — gray-50 → gradient (white → purple-50)
3. **메인페이지 카드 간격** — gap-2 → gap-3 sm:gap-4
4. **반응형 간격 세분화** — py-12 md:py-20 → py-8 sm:py-12 md:py-16 lg:py-20
5. **상태 뱃지 통일** — 모든 상태 뱃지를 50/700 톤으로 통일
6. **텍스트 대비율 확보** — text-gray-300/400 → text-gray-500/600 (WCAG AA)

### 2단계: 컴포넌트 업그레이드 (소요: 1~2일)
> 무료 라이브러리에서 패턴을 참고하여 우리 스타일로 재구현

**작업 목록:**
1. **Hero 섹션 리디자인** — Magic UI 참고, Framer Motion 강화
2. **카드 컴포넌트 업그레이드** — 호버 효과, 그림자 체계 정비
3. **사장님 대시보드 UI 현대화** — Origin UI 대시보드 패턴 참고
4. **통계 카드 리디자인** — 아이콘 + 숫자 + 트렌드 화살표
5. **모바일 UX 최적화** — 터치 타겟 48px, iOS 스크롤 개선

### 3단계: 전체 디자인 시스템 정리 (소요: 1일)
> CSS 변수 통일 + globals.css 정리

**작업 목록:**
1. **색상 CSS 변수 완전 통일** — 하드코딩 → var() 전환
2. **타이포그래피 유틸리티 정리** — h1~h5 + body + caption 클래스
3. **간격 시스템 점검** — 8px 그리드 위반 모두 수정
4. **그림자 체계 정비** — xs/sm/md/lg/xl 5단계 통일
5. **트랜지션 통일** — fast/normal/slow 3단계

## 5. Figma 연합작전 (나중에 필요할 때)

### Figma 무료 플랜으로 가능한 것
- 3개 파일 + 무제한 Draft
- Community에서 무료 UI 키트 4,770개+
- 디자인 시안 작성 → 대장님이 직접 확인 가능

### Figma MCP (Claude Code 연동)
- Figma 공식 MCP 서버 존재: `https://mcp.figma.com/mcp`
- 설정: `claude mcp add --transport http figma https://mcp.figma.com/mcp`
- **제약**: Dev Mode 필요 (유료) — 무료 플랜에서는 제한적
- **판단**: 지금은 불필요. 디자이너 합류 시 도입

### Penpot (오픈소스 대안)
- 완전 무료, 무제한, 자체 호스팅 가능
- Figma 95% 유사 인터페이스
- **판단**: MAU 1만 넘으면 그때 검토
