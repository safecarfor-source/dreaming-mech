# 커뮤니티 활성화 전략

## 상태: 기획 완료 — 글쓰기 시점에 재논의

## 글 분류
1. **정보전달** — 정비 꿀팁, 차종별 주의사항, 계절 정비, 타이어/오일 등
2. **가치입증** — 네이버 블로그 벤치마킹 예정 (대장님이 예시 공유 예정)

## 글 구조 (템플릿)
- 카테고리 뱃지 (꿀팁/질문/차종별/계절정비/사기방지)
- 섹션1: 핵심 요약 (2~3줄, 결론 먼저)
- 섹션2: 상세 설명 (사진/일러스트)
- 섹션3: 관련 유튜브 영상 임베드
- 하단 CTA: "가까운 정비소 찾기"

## 시딩 주제 7개 (초기)
1. 엔진오일 교환주기, 5,000km vs 10,000km 정답은?
2. 정비소에서 이 말 하면 바가지입니다
3. 소나타 10만km 넘으면 반드시 점검할 5가지
4. 여름 에어컨 점검, 이것만 확인하세요
5. 타이어 교체시기, 이렇게 확인하면 됩니다
6. 중고차 구매 전 정비 이력 확인하는 법
7. K5 오너라면 알아야 할 정비 포인트

## 스케줄
- 주 2개 작성 (월/목)
- Claude 초안 → 대장님 검수 → 게시

## 작업 흐름
1. Claude가 유튜브 영상 기반 주제 선정 + 초안 작성
2. 대장님에게 알림 (세션 시작 시 보고)
3. 대장님 검수/수정
4. 게시

## 정비소 대표님 글 전략
- 정비소 소개글
- 정비 사례 (차종+증상+해결)
- 지역 정비 팁
- "정비소 인증" 뱃지 표시

## 3단계 활성화
1. 시딩 (30개 글 미리 채우기)
2. 트리거 (유튜브 영상 → 커뮤니티 연결)
3. 루프 (질문→답변→알림→재방문)

## 탭바 확장 조건
- 글 80개+ 달성 시 하단 탭바에 💬 커뮤니티 탭 추가
