# 커뮤니티 글쓰기 가이드 — 가치입증 편

## 상태: 글쓰기 시점에 활용

---

## 글 2대 분류

| 분류 | 목적 | 예시 |
|------|------|------|
| **정보전달** | SEO 유입 + 전문성 증명 | 엔진오일 교환주기, 타이어 교체시기 |
| **가치입증** | 신뢰 구축 + 전환 유도 | 성과 증거, 고객 후기, 플랫폼 가치 |

---

## 가치입증 글 공식 (벤치마킹 분석 결과)

### 5단계 구조

```
1단계: 공감 훅       → "이런 고민 있으시죠?" (독자의 아픔 건드리기)
2단계: 반전          → "근데 누군가는 이걸로 성과 내고 있습니다"
3단계: 증거          → 실제 성과 숫자, 후기, 스크린샷 (캡처 필수)
4단계: 방법론 3가지   → 핵심 노하우 1,2,3 (실행 가능한 수준)
5단계: 마무리 CTA    → "지금 시작하세요" + 연락처/링크
```

### 핵심 원칙
- **숫자가 신뢰를 만든다** — "10개월", "90%", "1,400명", "52,000명"
- **대화체로 쓴다** — 딱딱한 설명 X, 말하듯이 써야 45세+ 남성이 읽음
- **증거 = 스크린샷** — 말만 하지 않고 캡처로 보여줌
- **방법론은 항상 3가지** — 많으면 안 읽음, 적으면 부실
- **중간중간 여백** — 한 문단 2~3줄 이하, 모바일에서 읽기 편하게

## 벤치마킹 글 3개 분석

### 글1: "블로그 망했다고요?" (네케팅)

| 단계 | 내용 |
|------|------|
| 훅 | "네이버 블로그 망했다", "요즘 블로그 누가보냐" — 공감 |
| 반전 | "누군가는 항상 성과를 독점하고 있고, 블로그만으로도 T.O를 항상 조기 마감" |
| 증거 | "9월 마감" "10월 한 팀" "11월 마감" 스크린샷 + "거짓 시 1억 배상" 각서 |
| 방법론 | 1. 지역 키워드를 먹으세요 (정보성 X, 지역 O) |
|  | 2. 최소 3~6개월 하세요 (단기 성과 기대 금지) |
|  | 3. 다른 SNS도 키우세요 (블로그+스레드 권장) |
| CTA | "지금 이야기한 것들만 바로 실행하셔도 3개월 후 완전히 다른 사업체" |

**배울 점:**
- "조회수 100 나오면 문의가 오나요?" 같은 직접적 질문 → 독자 스스로 답하게 만듦
- 지역 키워드 = 우리 정비소 플랫폼과 직결 (지역 SEO)
- "3~6개월" 타임라인 제시 → 현실적 기대치 설정

### 글2: "대치동 학원 부럽지 않은 신규생 마케팅" (네케팅)

| 단계 | 내용 |
|------|------|
| 훅 | "매 학기마다 홍보물 뿌리고 설명회 열어도 신규생 등록이 예전 같지 않죠?" |
| 반전 | "5년 동안 수많은 교육 업체의 성공을 이끈 마케터로서 단언컨대..." |
| 증거 | 구체적 업종(학원) + 5년 실적 언급 |
| 방법론 | 1. 학부모는 성적이 아니라 변화된 미래를 검색 (키워드 전략) |
|  | 2. 실생활이 담긴 모습이 더 먹힌다 (현장감 콘텐츠) |
|  | 3. 전략 없으면 의미 없다 (시즌별 그물망 전략) |
| CTA | "원장님은 수업에만 집중, 마케팅은 저희가" |

**배울 점:**
- **업종 특화** — "학원"처럼 타겟 명확, 우리도 "정비소"로 특화
- **감정 터치** — "실력 있는 원장님이 홍보 부족으로..." → "실력 있는 정비소가 홍보 부족으로..."
- **역할 분리** — "당신은 본업, 우리는 마케팅" 구조

### 글3: "이상한마케팅 원데이부트캠프 4주 후기" (고객 후기형)

| 단계 | 내용 |
|------|------|
| 훅 | "솔직히 처음엔 주저했습니다. 하루 99만원." |
| 반전 | "4주가 지난 지금은 그 생각이 완전히 바뀌었습니다" |
| 증거 | 4주 14개 포스팅, 체험 상담 등록률 90%, 키워드 1페이지 노출 |
| 방법론 | 1. 매니저님들의 진심 (사람에 대한 신뢰) |
|  | 2. 실행하게 만든다 (차별점) |
|  | 3. 지금 당장, 오늘 바로 쓰게 만든다 (즉시성) |
| CTA | "언젠가 성공사례로 소개되고 싶다" (미래 비전) |

**배울 점:**
- **솔직한 주저함으로 시작** — "처음엔 비싸다고 생각" → 공감
- **구체적 숫자로 전환** — "4주 14개", "90%", "1페이지"
- **사람 이야기** — 시스템이 아니라 사람에 대한 신뢰 → 우리도 "20년차 정비사"

## 우리 플랫폼에 적용 — 가치입증 글 템플릿

### 템플릿 A: 플랫폼 가치입증

```
[훅]
"정비소 찾을 때 네이버 후기만 보시나요?
그거 광고 아닌 거 확인하셨어요?"

[반전]
"20년차 정비사가 직접 발로 뛰며 검증한 정비소만 모았습니다."

[증거]
- 유튜브 구독자 52,000명
- 검증된 정비소 12개
- 고객 문의 847건
- (스크린샷 첨부)

[방법론]
1. 정비사 경력 20년이 검증하는 법
2. 이런 정비소는 피하세요 (체크리스트)
3. 가까운 검증 정비소 찾는 법

[CTA]
"내 근처 검증된 정비소 찾기 → dreammechaniclab.com"
```

### 템플릿 B: 정비소 성공사례

```
[훅]
"정비소 사장님, 손님이 줄었다고 느끼시죠?"

[반전]
"꿈꾸는정비사에 등록한 ○○정비소는 월 문의가 X건 늘었습니다."

[증거]
- 등록 전/후 문의 수 비교
- 고객 후기 캡처
- 정비소 사장님 인터뷰

[방법론]
1. 유튜브 노출 효과 (구독자 52K)
2. 지역 검색 상위 노출
3. 검증 마크의 신뢰 효과

[CTA]
"정비소 등록 문의 → dreammechaniclab.com/pro"
```

### 템플릿 C: 고객 후기형

```
[훅]
"솔직히 인터넷으로 정비소 찾는 거 불안했습니다."

[반전]
"근데 여기는 유튜버가 직접 검증한 곳이라 한번 가봤는데..."

[증거]
- 실제 정비 후기 (비용, 소요시간, 만족도)
- 정비소 사진
- 비교 견적 (다른 곳 vs 여기)

[방법론]
1. 어떻게 찾았는지 (검색 경로)
2. 왜 여기를 선택했는지 (신뢰 요소)
3. 결과가 어땠는지 (만족도)

[CTA]
"나도 검증된 정비소 찾기 →"
```
