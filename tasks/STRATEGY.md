## 3. 성장 플라이휠 (이것만 돌리면 된다)

```
유튜브 영상 (52K 구독자)
    ↓
소비자 유입 → 문의 접수 (전화번호)
    ↓
텔레그램 알림 → 단톡방(600+) 공유
    ↓
정비사 가입 → 고객 전화번호 확인 → 전화 연결
    ↓
정비 완료 → 만족 → 재이용 + 입소문
    ↓
더 많은 소비자 → 더 많은 정비사 가입 (양면 시장 효과)
```

**핵심 병목:** 소비자 문의가 충분해야 정비사가 가치를 느낀다
**해결책:** 유튜브 영상마다 플랫폼 링크 노출 → 문의 건수 확보

## 4. Build-Measure-Learn 실행 사이클

우리는 2주가 아니라 **1~3일** 단위로 돌린다.

```
[Build] 기능 구현 (Claude Code, 몇 시간)
    ↓
[Deploy] 로컬 확인 → 서버 배포 (30분)
    ↓
[Measure] 실사용 데이터 확인 (관리자 대시보드)
    ↓
[Learn] 피드백 (유튜브 댓글, 카톡 단톡방, 직접 사용)
    ↓
[Build] 다음 개선...
```

## 7. 유튜브 ↔ 플랫폼 시너지 전략

대장님의 최대 무기는 유튜브 채널. 이걸 플랫폼과 직접 연결해야 한다.

### 영상 유형별 플랫폼 연결
| 유튜브 영상 | 플랫폼 연결 | 기대 효과 |
|------------|-----------|----------|
| "이런 정비소 가지 마세요" | 영상 끝에 "믿을 수 있는 정비소 찾기 → 플랫폼 링크" | 소비자 유입 |
| "정비사 수입 현실" | "고객을 직접 찾을 수 있는 플랫폼 → 가입 링크" | 정비사 유입 |
| "내가 플랫폼을 만든 이유" | 전체 스토리 + 가입 유도 | 양쪽 유입 |
| 정비 과정 영상 | "이 정비소도 플랫폼에 등록되어 있어요" | 신뢰 구축 |

### 추적 링크 활용
- 영상별 TrackingLink 생성 (예: `?t=youtube_tire_video_01`)
- 어떤 영상에서 가장 많이 유입되는지 측정
- 효과 좋은 영상 유형에 집중

- [ ] 정비사 피드백 (단톡방, 카톡)에서 불편한 점은?
- [ ] 소비자 피드백 (유튜브 댓글)에서 원하는 기능은?

## 9. 절대 하지 말 것

1. **기능 100개 brainstorm** — 1가지를 완벽하게 하는 게 10가지 대충보다 낫다
2. **외주 개발** — AI 개발 체제가 더 빠르고 저렴하다
3. **투자 유치** — 유튜브 수익 + 서비스 수익으로 자생 가능
4. **앱 개발** — 웹이 먼저. 모바일 웹으로 충분. 앱은 MAU 1만 넘으면 그때
5. **완벽주의** — 80% 완성 → 배포 → 피드백 → 개선이 100% 완성보다 빠르다

## 10. 전쟁 타임라인 (2026 상반기)

### 현재 전력 (2026-02-26 기준)
| 구분 | 점수 | 평가 |
|------|------|------|
| 기술 준비도 | **88%** | 핵심 기능 거의 완성 + 디자인 통일 + 버그 수정 |
| 비즈니스 준비도 | **35%** | 실사용자 유입이 최대 병목 (커뮤니티 Q&A로 개선 예정) |
| 조직 준비도 | **90%** | 창업자 + AI 개발 체제 최적 |

### 작전 스케줄
```
[Week 1] 2026-02-26 ~ 03-04
  ├─ ✅ 웹 디자인 세련화 (퍼플 통일 + 폼 모던화 완료)
  ├─ ✅ 버그 수정 (영업시간 + 문의 모달)
  ├─ 커뮤니티 Q&A 구현 (플랜 승인, 착수 대기)
  └─ 유튜브 플랫폼 소개 페이지 + TrackingLink

[Week 2~3] 2026-03-05 ~ 03-18
  ├─ 첫 유튜브 영상 배포 + 반응 측정
  ├─ 정비사 마이페이지 + 문의 알림
  └─ SEO 메타태그 최적화

[Week 4~6] 2026-03-19 ~ 04-08
  ├─ 유튜브 데이터 기반 영상 최적화
  ├─ 정비소 상세 페이지 SEO
  └─ 카카오 알림톡 자동 발송

[Month 3~4] 2026-04 ~ 05
  ├─ 월 100건 문의 + 정비사 50명 달성
  └─ 수익화 준비 (구독 결제 시스템)

[Month 5+] 2026-06~
  ├─ Phase B 시작 (월 30만원 구독)
  └─ 정비사 50명 × 30만원 = 월 1,500만원
```

### P0 작전 (최우선): 유튜브 연동
기능은 다 만들었다. **총알(고객)이 없다.**
52K 유튜브 채널 = **유일한 무료 마케팅 무기**
→ 유튜브 영상 1개가 코드 100줄보다 가치 있는 시점

### P1 작전: SEO + 자연 유입
유튜브 외 검색 유입을 확보해야 지속 가능
→ "서울 정비소 추천", "부산 자동차 정비" 등 지역 키워드

### P2 작전: 정비사 경험 고도화
정비사가 떠나지 않게 만들어야 플라이휠이 돈다
→ 마이페이지, 알림, 대시보드
