# 절대 하지 마라 목록

## DB
- `prisma db push` (프로덕션)
- `prisma migrate reset` (프로덕션)
- WHERE 조건 없는 DELETE
- app_user로 ALTER TABLE
- 백업 없이 마이그레이션 실행

## 코드
- 유사 퍼플 색상 사용 (#7C4DFF만 허용, hover: #6B3FE0)
- API 응답 필드 삭제/이름변경
- 비밀번호 평문 저장
- JWT fallback secret (`|| 'fallback-secret'`)
- innerHTML + 사용자 입력
- 같은 유틸 함수 2곳 이상 복사 (공통화 필수)
- 상수 2곳 이상 정의 (단일 소스 필수)

## 프로세스
- 빌드 검증 없이 완료 표시
- 마이그레이션 파일 사후 추가 (이미 서버 적용된 것)
- 기존 코드에 불필요한 주석 추가
- 과잉 설계 (미래 대비 추상화)
- 비밀번호/계정 정보 임의 변경
- 대장님 말 끝나기 전에 실행 시작

## 아키텍처
- API 응답에 비밀번호 필드 포함
- N+1 쿼리 방치
- limit=1000 하드코딩 (페이지네이션 필수)
- 루프 안에서 DB 호출
